ENTORNO DE DESARROLLO
---------------------
ECLIPSE KEPLER
JAVA 1.8
MAVEN 3
TOMCAT 7

PRE REQUISITOS
--------------
CREAR UNA BASE DE DATOS EN MYSQL
LANZAR EL SCRIPT INSTALL.SQL PARA CREAR LA TABLA Y CARGARLA CON 150K REGISTROS
CONFIGURAR EL ARDHIVO APPLICATION.PROPERTIES CON LAS CONEXIONES A LA BASE DE DATOS
COMPILAR CON EL COMANDO : mvn clean install

OBSERVACIONES
-------------
EL FILTRADO DE LA TABLA HACE BÚSQUEDA COMPLETAS DE NOMBRES EN CUALQUIERA DE SUS CAMPOS (SI UN USUARIO SE LLAMA JUAN ANTONIO, AL ESCRIBIR JUAN NO LO ENCONTRARÁ)
